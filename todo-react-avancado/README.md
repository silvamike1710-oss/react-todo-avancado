# Todo React Avançado

## Descrição

Aplicação de lista de tarefas desenvolvida com React utilizando conceitos avançados da biblioteca.

## Tecnologias utilizadas

* React
* Vite
* Context API
* Hooks
* Custom Hooks
* Memoization

## Funcionalidades

* Adicionar tarefas
* Marcar tarefas como concluídas
* Remover tarefas
* Filtrar tarefas (todas, concluídas, pendentes)
* Persistência com localStorage

## Como rodar o projeto

```bash
npm install
npm run dev
```

Depois acesse:

```
http://localhost:5173
```
